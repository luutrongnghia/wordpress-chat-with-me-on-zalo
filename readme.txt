== Installation ==

=== From within WordPress ===

1. Visit 'Plugins > Add New'
2. Search for 'Chat With Me on Zalo'
3. Activate Chat With Me on Zalo from your Plugins page.

=== Manually ===

1. Upload the `chat-with-me-on-zalo` folder to the `/wp-content/plugins/` directory
2. Activate the Chat With Me on Zalo plugin through the 'Plugins' menu in WordPress
3. Go to "after activation" below.
